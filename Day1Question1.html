Question 1

Explore and explain the various methods in console function Explain them Ex. console.log() console.warn(). etc...

console : The console object provides access to browsers debuggin mode
console.time() : Starts a timer (can track how long an operation takes)
console.log() : Outputs a message to the console
console.assert() : Writes an error message to the console if the assertion is false
console.clear() : Clears the console
console.table() : Displays tabular data as a table
console.error() : Outputs an error message to the console
console.warn() : Outputs a warning message to the console
console.timeEnd(): Stops a timer that was previously started by console.time()
